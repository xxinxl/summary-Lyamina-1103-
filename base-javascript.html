Арифметические операции Описание Пример Результат
Сложение(+) Складывает два операнда 3 + 5 8
Вычитание(-) Вычитает правый операнд из левого 7 - 2 5
Умножение (*) Умножает два операнда 4 * 6 24
Деление(/) Делит левый операнд на правый 8 / 2 4

Функции lodash Описание Пример
_.map() Преобразует каждый элемент коллекции с использованием переданной функции _.map([1, 2, 3], function(n) { return n * 2; })
_.filter() Отфильтровывает элементы коллекции на основе предиката _.filter([1, 2, 3, 4], function(n) { return n % 2 === 0; })
_.reduce() Сводит коллекцию к значению, используя аккумулятор и функцию _.reduce([1, 2, 3], function(sum, n) { return sum + n; }, 0)
_.sortBy() Сортирует элементы коллекции на основе указанных ключей или функций _.sortBy([{ 'name': 'John' }, { 'name': 'Doe' }], 'name')
_.debounce() Создает функцию-декоратор, которая откладывает вызов функции до тех пор, пока проходит указанный интервал времени между ее вызовами var debounced = _.debounce(func, 2000)
_.clone() Создает копию значения, рекурсивно копируя все свойства var objects = [{ 'a': 1 }, { 'b': 2 }]; var deep = _.cloneDeep(objects);
_.isEqual() Сравнивает два объекта на равенство. Рекурсивное сравнение _.isEqual(object, other)
_.memoize() Мемоизация результата выполнения функции для ускорения последующих вызовов с теми же аргументами var memoized = _.memoize(func)
Логические операции Описание Пример Результат
Логическое И (&&) Возвращает true, если оба операнда истинны (5 > 3) && (10 < 20) true
Логическое ИЛИ (||) Возвращает true, если хотя бы один из операндов истинен (5 > 3) || (10 > 20) true
Логическое НЕ (!) Инвертирует значение операнда (если истина, то становится ложью, и наоборот) !(5 > 3) false
Логическое И с присваиванием (&&=) Присваивает второй операнд первому, если оба истинны let a = true; a &&= (10 < 20)
Логическое ИЛИ с присваиванием (||=) Присваивает второй операнд первому, если хотя бы один из них истинен let b = false; b ||= (10 > 20) b = false
Тернарный оператор (?:) Возвращает значение в зависимости от условия let x = (5 > 3) ? 'Да' : 'Нет' x = 'Да'
Алгоритмические конструкции Описание
Условие (if-else) Выполняет блок кода, если условие истинно, и блок кода в противном случае
Цикл while Повторяет блок кода, пока условие истинно
Цикл do-while Повторяет блок кода, пока условие истинно, хотя бы один раз
Цикл for Повторяет блок кода с предопределенным числом итераций
Цикл for...in Итерируется по перечисляемым свойствам объекта
Цикл for...of Итерируется по итерируемым объектам (массивам, строкам, и т. д.)
Функции Способ объявления Пример использования
Обычная функция (function declaration) Объявляется с использованием ключевого слова function. Например, function sum(a, b) { return a + b; } Вызывается, например, как sum(2, 3), что вернет 5
Функция-выражение (function expression) Функция присваивается переменной. Например, const multiply = function(a, b) { return a * b; } Вызывается, например, как multiply(2, 3), что вернет 6
Стрелочная функция (arrow function) Используется синтаксис с стрелкой =>. Например, const square = (x) => { return x * x; } Вызывается, например, как square(4), что вернет 16
